# WezTerm key-bindings
`Leader Key: ctrl + space`
## General
`ctrl+p: comand palet`  
`ctrl+shift+c: copy selection`  
`ctrl+shift+v: paste selection`  
## Tabs & panes
`alt+q: quite pane`  
`alt+f: Fullscrean pane`  
`alt+DownArrow: create pane below`  
`alt+RightArrow: create pane Right`  
`leader,n: new tab`  
`alt+m: tabs right`  
`alt+n: tabs left`
